Handle EMSL Gaussian basis set files.

This module reads EMSL basis set files in Gaussian94 format. These files may be downloaded from the Gaussian Basis Set Order Form run by the Environmental and Molecular Science Laboratory at the Pacific Northwest Laboratories.
