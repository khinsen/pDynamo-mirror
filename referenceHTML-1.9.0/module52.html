Python Module EnergyModel

Owner: Package pMolecule

Source File: pMolecule-1.9.0/pMolecule/EnergyModel.py

Defines the energy model for a system.

The model contains all the data and definitions necessary for an energy calculation apart from those that are stored in the system's configuration.

The order of definition of models is MM before QC before NB. Softconstraints are independent and can be defined at any time.
