Python Module RingFinder

Owner: Package pCore

Source File: pCore-1.9.0/pCore/RingFinder.py

Ring-finding module using Figueras's SSSR algorithm.

This module finds the smallest set of smallest rings (SSSR) for a given network based upon the algorithm of J. Figueras "Ring Perception Using Breadth-First Search" J. Chem. Inf. Comput. Sci. 36, 986-991 (1996).

The module works for 2-D networks (either in a plane or a surface in 3-D) and for networks that have at least some nodes with connectivities of 3 or less.

The connectivity table that is input to FindSSSR must be valid which means that it consists of a list of N nodes (numbered from 0 to N-1) with no nodes of other index appearing in the table. All connections are specified twice (i.e. N1-N2 is specified as N2 in the table for N1 and vice-versa).

The number of rings should equal ( Nring_bonds - Nring_atoms + 1 ) for each disconnected set of rings.
